# ⛏️ 矿工大亨 - 放置点击游戏

一个简单好玩的放置类网页游戏，目标是成为最富有的矿工！

## 🎮 游戏玩法

- 点击挖掘按钮获取金币
- 购买升级提升点击收益和自动收益
- 解锁成就获取更多奖励
- 每天登录领取奖励

## 🚀 部署

本游戏使用纯 HTML/CSS/JS 开发，可以部署到任何静态托管服务：

- GitHub Pages
- Netlify
- Vercel
- 任意 Web 服务器

## 📁 文件结构

```
idle-clicker/
├── index.html    # 游戏主文件
└── .nojekyll     # GitHub Pages 配置
```
